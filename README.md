# Innovation Expo 2026 — IEEE WIE

Official landing page for **Innovation Expo 2026**, organized by IEEE Women in Engineering (WIE), Bennett University.

> "Where Ideas Meet Impact" — 29 March 2026

---

## 🚀 Live Site

[innovation-expo-virid.vercel.app](https://innovation-expo-virid.vercel.app/)

---

## 📌 About

Built an Event Website for IEEE WIE's Innovation Expo 2026, which is a platform where students present real projects and pitch their ideas in a TEDx-style format to industry experts from companies like **Samsung** and **Ixigo**. Participants receive constructive feedback, gain professional exposure, and compete for recognition.

## ⚙️ Getting Started

### Prerequisites
- Node.js v18+
- npm

### Installation

```bash
git clone https://github.com/kushalaroraa/Innovation-Expo.git
cd Innovation-Expo
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```
